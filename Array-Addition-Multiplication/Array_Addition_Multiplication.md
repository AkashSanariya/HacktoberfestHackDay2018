# Array Row Addition and Column Multiplication

Defination: Create two input fields one is the number of rows and second is the number of columns using this input values **create a multidimensional random value array**.
# Example - 1:
Input:
  - NUmber of Rows : 5 (input by user)
  - NUmber of Columns : 5 (input by user)


Output:

[10 25  10  11  5]<br>
[5  63  12  44  4]<br>
[7  10  17  78  20]<br>
[62  88  12  44  33]<br>
[0  5  48  56  14]<br>

Result: 

  Row:1 ->  10+25+10+11+5 = 60<br>
  Row:2 ->  5+63+12+44+4 = 128<br>
  Row:3 ->  7+10+17+78+20 = 132<br>
  Row:4 ->  62+88+12+44+33 = 239<br>
  Row:5 ->  1+5+48+56+14 = 123<br>
  
  Column:1 ->  10x5x7x62x0 = 0<br>
  Column:2 ->  25x63x10x88x5 = 6930000<br>
  Column:3 ->  10x12x17x12x48 = 1175040<br>
  Column:4 ->  11x44x78x44x56 = 93020928<br>
  Column:5 ->  5x7x20x33x14 = 323400<br>
